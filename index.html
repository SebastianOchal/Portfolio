Simon/Follow The Pattern

I developed this game primarily using JavaScript. The core gameplay involves following a given pattern. To increase the complexity of the project, I added a feature allowing users to resize the grid while the game is inactive. This presented several challenges, the most significant being that resizing the grid initially caused it to stop responding to user inputs. Resolving this issue required considerable effort and troubleshooting.

Portfolio

This portfolio stands as a showcase of some of my most advanced HTML, CSS, and JavaScript skills learned over an 8-week bootcamp provided by Just IT. One part of the portfolio I am most proud of is the background, which is made entirely using HTML elements with CSS styling applied. A big challenge I faced was that CSS is very good at creating regular patterns but not so good at generating random patterns. As such, I had to use JavaScript to position the stars within my portfolio. This solution made my portfolio more dynamic, as the stars change position when the page is refreshed.
